# Lab 3: Bob & ADK Setup

## Overview

This lab covers setting up Bob (IBM's AI coding assistant) and the watsonx Orchestrate Agent Development Kit (ADK) for building custom agents and tools.

> **📥 Download Setup Guide:** [IBM i User Group - WxO Bob Setup.pptx](./assets/IBM%20i%20User%20Group%20-%20WxO%20Bob%20Setup.pptx)

## What You'll Learn

- Installing and configuring Bob in VS Code
- Setting up the watsonx Orchestrate ADK
- Configuring Python environment
- Connecting to watsonx Orchestrate instances
- Understanding the ADK project structure

## Prerequisites

- VS Code installed
- Python 3.11+ installed
- Access to watsonx Orchestrate instance
- GitHub account (for Bob authentication)

## Setup Instructions

Please refer to the PowerPoint guide above for detailed step-by-step instructions on:

1. Installing Bob extension in VS Code
2. Setting up Python virtual environment
3. Installing the watsonx Orchestrate ADK
4. Configuring environment variables
5. Authenticating with watsonx Orchestrate
6. Verifying your setup

## Next Steps

After completing this setup, proceed to:
- **Lab 4:** Build your first custom agent with Bob
- **Lab 5:** Create agentic workflows for expense processing
- **Lab 6:** Develop custom tools
